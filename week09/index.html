Notes

  • The Browser Object Model is a collection of properties and methods that contain information about the browser and computer screen.
  • Global variables can be accessed in all parts of the program and are properties of a global object.
  • Using popups is not recommended, you have to look for better alternatives.
  • The Screen object has more uses for mobile devices. It also allows you to do things like turn off the device's screen, detect a change in its orientation or lock it in a specific orientation.
  • Cookies are small files that are saved locally on a user's computer and can be used to store information that can then be retrieved between requests.
  • Cookies have many limitations, it is preferable to use LocalStorage, although cookies are still used, such as to keep a session open in a browser.
  • To remove a cookie, you need to set it to expire at a time in the past.
  • The window object provides some useful methods for scheduling the execution of a function, and for repeatedly executing functions at regular intervals.
  • setTimeout() method can be used to invoke a function after a set amount of time.
  • setInterval() method can be used to repeatedly invoke a function.
  • requestAnimationFrame() method can be used to produce smooth and animation optimized by utilizing the browser's built-in graphics capabilities.
